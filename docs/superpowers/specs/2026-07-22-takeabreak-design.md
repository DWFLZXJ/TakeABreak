# Take a Break — 产品设计规格

> 日期：2026-07-22  
> 状态：已确认（头脑风暴）  
> 平台：macOS  
> 方案：极简番茄（方案 1）

## 1. 产品概述

**Take a Break** 是一款 macOS 休息提醒工具：在设定的工作时长结束后，以全屏壁纸 + 自定义文字叠层提醒用户休息；休息结束后自动进入下一轮工作。

### 1.1 设计原则

- **简约扁平**：界面贴合系统浅色 / 深色主题，少装饰、少层级
- **壁纸优先**：休息瞬间以用户壁纸为主角，文字克制
- **轻打扰日常、重提醒节点**：日常只占菜单栏；到点全屏
- **YAGNI**：第一版只做完整番茄闭环，不做统计 / 云同步 / 多档案

### 1.2 已确认决策

| 决策点 | 选择 |
|--------|------|
| 产品形态 | **混合**：菜单栏日常控制 + 偏好设置独立小窗 |
| 休息视觉 | **纯文字叠层**（无卡片、无毛玻璃浮窗） |
| 跳过休息 | **长按 2 秒**（偏好可关；关则必须等倒计时） |
| 时长模型 | **两段式番茄**：工作 N 分钟 → 休息 M 分钟，循环 |
| 产品范围 | **极简番茄（方案 1）** |

## 2. 状态模型

### 2.1 状态

| 状态 | 含义 | 主要 UI |
|------|------|---------|
| `Idle` | 未运行 | 菜单栏静态图标；面板可「开始专注」 |
| `Working` | 工作倒计时中 | 菜单栏显示剩余 `mm:ss` |
| `Paused` | 工作暂停 | 菜单栏 `‖ mm:ss`（或等价弱化样式） |
| `Breaking` | 休息倒计时中 | **全屏**壁纸 + 文字叠层 |

### 2.2 主循环

```
Idle → Working ⇄ Paused
         ↓ (N 到点)
      Breaking ──(M 到点)──→ Working
         │
         ├─ 长按跳过(若允许) → Working
         └─ 停止 → Idle
Working / Paused ─ 停止 → Idle
```

### 2.3 规则

1. **工作到点**：自动进入 `Breaking`，不弹确认。
2. **休息到点**：自动进入下一轮 `Working`。
3. **暂停**：仅 `Working` 可进入 `Paused`；`Breaking` 不可暂停。
4. **停止**：任意运行态可回 `Idle`；若在 `Breaking`，先退出全屏。
5. **改时长 N / M**：只影响**下一轮**；当前轮在进入该状态时锁定时长。
6. **计时**：使用单调时钟；睡眠 / 合盖期间冻结，唤醒后按剩余时间继续（睡眠不计入工作）。

## 3. 界面规格

### 3.1 菜单栏图标

| 状态 | 显示 |
|------|------|
| Idle | 静态符号图标（无数字） |
| Working | 剩余 `mm:ss` |
| Paused | `‖ mm:ss` 或同款变淡 |
| Breaking | 可选显示休息剩余（全屏为主） |

### 3.2 菜单栏下拉面板

宽度约 **280px**，跟随系统外观。

**Idle**

- 摘要行：`工作 {N} 分 · 休息 {M} 分`
- 主按钮：**开始专注**
- 底栏：**偏好设置…**（可显示 ⌘,）

**Working**

- 标签「工作中」+ 可选「第 N 轮」
- 大号剩余时间 + 细进度条（系统蓝）
- 按钮：**暂停** | **停止**
- 底栏：**偏好设置…**

**Paused**

- 同 Working 布局；主操作改为 **继续**

**Breaking**（面板仍可打开时）

- 标签「休息中」+ 剩余时间
- **停止**（结束整轮 → Idle）
- **不提供**面板内「跳过」——跳过仅全屏长按

### 3.3 偏好设置小窗

约 **420×520**，单页 inset 分组表单，改完即时写入本地偏好（无大「保存」按钮）。

| 分组 | 内容 | 约束建议 |
|------|------|----------|
| 时长 | 工作 N、休息 M | 工作 5–90 分；休息 1–30 分；默认 25 / 5 |
| 休息壁纸 | 缩略图选择；内置默认 +「+」选本地文件 | 当前选中描边高亮 |
| 休息文案 | 多行文本 | 默认：「站起来走走，看看远处」；空则回退「该休息一下了」 |
| 行为 | 开关：允许长按跳过休息 | 默认开；关闭后无跳过 UI |

**生效时机**

- N / M：下一轮
- 壁纸 / 文案：默认**下一次进入休息**时生效（实现简单、可预期）

### 3.4 全屏休息（签名界面）

**视觉：纯文字叠层（无卡片）**

层级（底 → 顶）：

1. 用户壁纸 full-bleed cover  
2. 可读性层：轻微暗角和/或 20–40% 黑色透明叠层  
3. 小标签：`休息中`（宽字距、半透明）  
4. 自定义文案（细字重、居中；约 2–3 行）  
5. 倒计时 `mm:ss` + 细进度线  
6. 底部：长按跳过控件（若偏好开启）

**长按跳过**

| 项 | 规格 |
|----|------|
| 时长 | 2 秒 |
| 反馈 | 圆环 0→100%；松手未满回 0 |
| 成功 | 淡出全屏 → `Working` |
| 偏好关 | 不显示跳过 UI |

**动效**

- 进入 / 离开：0.4–0.6s 淡入淡出；无弹跳、无夸张动效
- 单击、Esc：默认**无效**（防误触）

**多显示器**

- 每屏铺同一壁纸与同一文字层（内容居中）

## 4. 数据与持久化

### 4.1 用户偏好（本地）

```
workMinutes: number          // 默认 25
breakMinutes: number         // 默认 5
wallpaperId: string          // 内置 id 或 local file bookmark/path
customMessage: string        // 休息文案
allowLongPressSkip: boolean  // 默认 true
```

### 4.2 运行时状态（可不持久化，或持久化以便崩溃恢复）

```
phase: Idle | Working | Paused | Breaking
roundIndex: number           // 从 1 起，可选展示
phaseEndsAt or remainingMs   // 与单调时钟配合
lockedWorkMinutes            // 本轮锁定
lockedBreakMinutes
```

第一版可不做崩溃恢复；重启后回 `Idle` 可接受。

## 5. 边缘情况

| 场景 | 处理 |
|------|------|
| 睡眠 / 合盖 | 计时冻结；唤醒继续剩余时间 |
| 系统时间被改 | 用单调时钟，避免跳变 |
| 壁纸文件缺失 | 回退内置默认；设置中可标失效 |
| 文案为空 | 默认「该休息一下了」 |
| 工作中改偏好 | 允许；N/M 仍只作用于下一轮 |
| 休息中停止 | 退全屏 → Idle |

## 6. 明确不做（v1）

- 今日统计、成就、每 4 轮长休息
- 多条文案轮换、多套预设档案
- 云同步、账号
- 自定义全局快捷键（后续可加）
- 无法退出的硬锁模式

## 8. 成功标准

1. 用户能在 30 秒内完成：设时长 → 开始 → 等到/模拟休息全屏 → 理解长按跳过  
2. 休息屏：壁纸清晰、文案可读、无卡片框  
3. 设置与菜单栏在浅/深色下均清晰、扁平、不喧宾夺主  
4. 番茄循环可连续多轮无手动重置（除非用户停止）
